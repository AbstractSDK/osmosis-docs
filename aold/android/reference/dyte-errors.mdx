# MeetingErrorListener

`MeetingErrorListener` is an interface that allows you to override different errors that can occur from within the client SDK, apart from exceptions.

```java
public void meetingNotFound();
public void unauthorized();
```

| Error           | Meaning                                                                                  |
| --------------- | ---------------------------------------------------------------------------------------- |
| unauthorized    | The `authToken` provided during initialization is incorrect / does not match our records |
| meetingNotFound | The `roomName` provided during initialization is incorrect / does not match our records  |
