# DyteMeeting

## Properties

| Property name       | Data type                                           | Use                                                                                                                                                                                  |
| ------------------- | --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| config              | [MeetingConfig](./connection-config)                | Connection info about the current meeting in progress, as defined below                                                                                                              |
| participants        | ArrayList <[DyteParticipant](./participant)>        | Array of participants, including the current participant (see [participant controls and actions](./../advanced-usage#controlling-individual-participants-and-actions) for more info) |
| self                | [DyteSelfParticipant](./self-participant)           | The current participant (see [self participant controls](./../advanced-usage#replace-the-meeting-control-buttons-with-your-own-buttons) for more info)                               |
| backgroundBehaviour | [BackgroundBehaviourType](#backgroundbehaviourtype) | Defines the meeting view behaviour for when the app is moved to background                                                                                                           |

## Methods

| Method name         | Params (with data type)                                                          | Use                                                                                                                                                                                        |
| ------------------- | -------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| setup               | config: [MeetingConfig](./connection-config)                                     | Setup a Dyte meeting with given config parameters                                                                                                                                          |
| updateUIConfig      | uiConfig: String (which is stringified JSON of [DyteUIConfig](./dyte-ui-config)) | Change the UI and layout of the current meeting according to the relevant params specified in the object (see [customize the meeting UI](./../customize-meeting-ui) for available options) |
| init                | activity: Activity, width: Integer, height: Integer                              | Initialize a view of given `width` and `height` with Dyte meeting, as a part of view heirarchy under the given `activity`                                                                  |
| addEventListener    | eventListener: [MeetingEventListener](./dyte-meeting-events)                     | Add an event listener to the meeting                                                                                                                                                       |
| removeEventListener | eventListener: [MeetingEventListener](./dyte-meeting-events)                     | Remove an event listener from the meeting                                                                                                                                                  |

## Interfaces

| Interface                                     | Purpose                                                    |
| --------------------------------------------- | ---------------------------------------------------------- |
| [MeetingEventListener](./dyte-meeting-events) | Helper interface for adding event listeners to the meeting |
| [MeetingErrorListener](./dyte-errors)         | Helper interface for adding error listeners to the meeting |

## BackgroundBehaviourType

| Behaviour Type          | Effect                                                                                                    |
| ----------------------- | --------------------------------------------------------------------------------------------------------- |
| PICTURE_IN_PICTURE      | Puts the meeting view in picture in picture mode, and continues with capturing the participant video      |
| DISABLE_CAMERA_CAPTURE  | Puts the meeting view in background (same as the app), and stops capturing participant video              |
| CONTINUE_CAMERA_CAPTURE | Puts the meeting view in background (same as the app), but continues with capturing the participant video |
